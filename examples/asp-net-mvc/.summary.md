# ASP.NET MVC Example - Summary

This folder contains a complete example of integrating CandleStick session tracking into an ASP.NET MVC application.

## Files Created

### Documentation
- **QUICK-START.md** - Get up and running in 5 minutes
- **README.md** - Complete integration guide with all configuration options
- **TROUBLESHOOTING.md** - Solutions to common issues

### Configuration
- **appsettings.json** - Development configuration
- **appsettings.Production.json** - Production configuration

### Views
- **Views/Shared/_Layout.cshtml** - Main layout with CandleStick integration
- **Views/Home/Index.cshtml** - Example home page showing tracking in action

## Key Features

### ✅ Simple Integration
Just add 2 script tags to your _Layout.cshtml file

### ✅ User Identification
Automatically captures user info from ASP.NET Identity:
- User ID
- User name
- User email
- Custom claims

### ✅ Environment-Aware
Different configuration for development vs production

### ✅ Error Handling
Graceful fallback if tracking fails - never breaks your app

### ✅ Privacy-Focused
Passwords automatically masked, configurable masking for other fields

## Quick Integration

1. **Start CandleStick services:**
   ```bash
   cd server && npm run dev          # Terminal 1
   cd packages/viewer && npm run dev # Terminal 2
   ```

2. **Copy the script tags from `Views/Shared/_Layout.cshtml` to your layout file**

3. **Done!** Sessions will appear at http://localhost:3000
